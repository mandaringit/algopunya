# [LEETCODE] Game of Life

m \* n 2차원 배열에서 각 원소의 값 & 주변 8곳의 값에 따라 주어진 조건을 만족하도록 구현. (동시 발생 & 공간 복잡도 O(1))

### 접근

문제를 읽고 구현은 거의 그대로하면 된다. 다만 동시다발적으로 변한다는 점에서, 초기 상태를 모두 유지한 상황에서 변화할 부분을 잡아내고, 순회를 마친 다음 그 요소들을 다시 찾아 변환시켜야 한다.

순회하면서 변화할 요소들을 좌표와 변화시킬 값으로 따로 저장한 뒤 한번 더 돌던지 (공간O(M\*N)), 공간 복잡도를 낮추기 위해 기존 배열에 다르게 표기해서 (0,1 이외에 2,3으로 표기) 로직 자체엔 영향을 주지 않고 변경시키는 방법을 사용했다.
